# Decidim::Gallery

Decidim Gallery Module.

## Usage

Gallery will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-gallery"
```

And then execute:

```bash
bundle
```
